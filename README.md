# Reci

Reci is a social recipe platform (think Beli for recipes): browse a feed of real recipes, rate/save/like them, follow other cooks, and climb a leaderboard. It's a monorepo with two top-level folders: `backend/` (FastAPI + PostgreSQL API) and `frontend/` (Next.js App Router web app). See `architecture.md` for the system design and `tasks.md` for the build plan.
